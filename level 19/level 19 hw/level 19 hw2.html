Block Level

1st block level element

This is a p element, p standing for paragraph, here you can write essays and texts, it takes as much space as possible, starts on a new line and take all width there is from left to right.



2nd block level element

This is a header element, header element contains introductory content in it. you can have multiple of these tags just like many others.


3rd block level element

This is a footer element, its very similar and different from the header tag, its always at the bottom aka the end of the site and contains usually ythe information about the contents above it.


Inline Level



1st inline level

This is a strong element, it's similar to the b (bold) element, but unlike b tag, which justs adds a cosmetic effect, strong tag is used to mark important stuff so that screen readers announce it strongly.

2nd inline level


Whats shown above is the img tag, its used to add images to your site you can customise the image like changing its height or width and etc.
